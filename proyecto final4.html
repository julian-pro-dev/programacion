Conceptos basicos de JavaScript

Se explicaran conceptos basicos de javascript a continuacion:

Sintaxis del lenguaje JavaScript

1. Estructura if:

La estructura if se utiliza para ejecutar un bloque de código si se cumple una condición especificada. Puede ir acompañada de una o varias estructuras else if y una estructura else (opcional) para evaluar condiciones adicionales.

2. Estructura switch:

La estructura switch se utiliza para seleccionar una de las muchas opciones posibles basada en el valor de una expresión.

3. Operador ternario:

El operador ternario es una forma concisa de escribir una estructura if-else en una sola línea. Se utiliza para tomar una decisión entre dos valores posibles en función de una condición.

Estas son las estructuras de selección básicas en JavaScript. Puedes combinarlas y anidarlas según sea necesario para realizar diferentes evaluaciones y decisiones en tu código.
