Conceptos basicos de JavaScript

Se explicaran conceptos basicos de javascript a continuacion:

Sintaxis del lenguaje JavaScript

Operadores en JavaScript:

1. Operadores aritméticos:

- Suma: +

- Resta: -

- Multiplicación: *

- División: /

- Módulo: %

- Incremento: ++

- Decremento: --

2. Operadores de asignación:

- Asignación: =

- Asignación de suma: +=

- Asignación de resta: -=

- Asignación de multiplicación: *=

- Asignación de división: /=

- Asignación de módulo: %=

3. Operadores de comparación:

- Igualdad: ==

- Desigualdad: !=

- Estrictamente igual: ===

- Estrictamente desigual: !==

- Mayor que: >

- Menor que: <

- Mayor o igual que: >=

- Menor o igual que: <=

4. Operadores lógicos:

- Y lógico: &&

- O lógico: ||

- Negación lógica: !

5. Operadores de concatenación:

- Concatenación de cadenas de texto: +

Orden de evaluación:

En JavaScript, las expresiones se evalúan de acuerdo con las reglas de precedencia y asociatividad de los operadores. Algunas pautas generales son:

- Las operaciones aritméticas tienen mayor precedencia que las operaciones de asignación.

- Las operaciones de asignación tienen menor precedencia que las operaciones aritméticas y de comparación.

- Los operadores lógicos se evalúan de izquierda a derecha.

- Utiliza paréntesis para controlar el orden de evaluación cuando sea necesario.

Espero que esta información te sea útil para comprender los tipos de datos, los operadores y el orden de evaluación en JavaScript.
