Conceptos basicos de JavaScript

Se explicaran conceptos basicos de javascript a continuacion:

Sintaxis del lenguaje JavaScript

1. Bucle while:

El bucle while se utiliza para ejecutar un bloque de código mientras se cumpla una condición especificada.

2. Bucle do-while:

El bucle do-while es similar al bucle while, pero la condición se evalúa después de ejecutar el bloque de código, por lo que se asegura de que el bloque se ejecute al menos una vez.

3. Bucle for:

El bucle for se utiliza para ejecutar un bloque de código un número específico de veces. Consiste en tres partes: la inicialización, la condición y la expresión de incremento o decremento.

4. Palabras clave break y continue:

- La palabra clave break se utiliza para salir inmediatamente de un bucle.

- La palabra clave continue se utiliza para saltar una iteración y continuar con la siguiente.

Estas son las estructuras de repetición básicas en JavaScript. Puedes utilizarlas para ejecutar bloques de código repetidamente en función de una condición o un número específico de veces.
