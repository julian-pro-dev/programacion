Sintaxis del lenguaje JavaScript

Un método es una función que está asociada a un objeto en JavaScript. Los objetos en JavaScript pueden tener propiedades y métodos. Los métodos se utilizan para realizar acciones o realizar operaciones específicas en el objeto al que pertenecen.

Puedes acceder a un método utilizando la notación de punto, que consiste en escribir el nombre del objeto, seguido de un punto y el nombre del método.

Aquí hay algunos ejemplos de métodos incorporados en JavaScript:

1. Métodos para cadenas de texto (String):

- length: Devuelve la longitud de una cadena de texto.

- toUpperCase(): Convierte una cadena de texto a mayúsculas.

- toLowerCase(): Convierte una cadena de texto a minúsculas.

2. Métodos para arrays:

- push(): Agrega uno o más elementos al final de un array.

- pop(): Elimina el último elemento de un array y lo devuelve.

- join(): Une todos los elementos de un array en una cadena de texto, separados por un separador especificado.

Espero que esta información te sea útil para comprender y utilizar los métodos en JavaScript.
