Sintaxis del lenguaje JavaScript

Las funciones son bloques de c√≥digo reutilizables que pueden recibir datos (parametros) como entrada, realizar operaciones y devolver un resultado (valor de retorno). En JavaScript, las funciones se definen utilizando la palabra clave "function".

Las operaciones matematicas basicas se realizan utilizando los operadores aritmeticos estandar:

- Suma: +

- Resta: -

- Multiplicacion: *

- Division: /

- Modulo: %

Recuerda que JavaScript tambien proporciona funciones matematicas incorporadas, como "Math.sqrt()" para calcular la raaz cuadrada, "Math.pow()" para elevar un numero a una potencia, "Math.round()" para redondear un numero, entre otras.
